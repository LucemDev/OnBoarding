### Implementation

Add jitpack to repository
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

Add the dependency
```
implementation 'com.github.Lucem-Anb:onboarding:0.0.1'
```
